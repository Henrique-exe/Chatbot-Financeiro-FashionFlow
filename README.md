💳 FashionFlow — Chatbot Financeiro
Uma interface de autoatendimento inteligente e responsiva voltada para operações financeiras da marca FashionFlow. O projeto simula uma experiência real de chat, auxiliando usuários em fluxos de pagamento, consultas e reembolsos.

🚀 Funcionalidades
Processamento de Intenções: Sistema baseado em palavras-chave que identifica a necessidade do usuário (saudações, pagamentos, rastreio, etc.).

Fluxos de Pagamento: Cartões informativos e botões de ação para Pix, Crédito (até 12x), Débito e Pagamento Misto.

Interface Dinâmica: Botões de resposta rápida (chips) que guiam a conversa e facilitam a navegação.

Simulação de Chat Real: Inclui indicador de digitação ("typing") e encerramento de sessão.

Design Moderno: UI limpa e responsiva, com uso de variáveis CSS para fácil customização.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica do chat.

CSS3: Estilização avançada com animações de entrada e estados de carregamento.

JavaScript (Vanilla): Lógica de busca de intenções, manipulação de DOM e gerenciamento do fluxo de conversa.

📂 Como utilizar
Clone este repositório.

Abra o arquivo fashionflow_chatbot_financeiro.html em qualquer navegador moderno.

Interaja com o bot através do campo de texto ou clicando nos botões de sugestão.
